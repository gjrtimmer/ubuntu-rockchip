About

Ubuntu Rockchip provides bootable Ubuntu disk images for Rockchip ARM single-board computers (SBCs), actively maintained with fixes and new board support.

Supported SoCs

  • RK3588 / RK3588S / RK3588S2
  • RK3576
  • RK3566

Ubuntu Releases

  • Ubuntu 22.04 LTS (Jammy Jellyfish)
  • Ubuntu 24.04 LTS (Noble Numbat)

Flavors

  • Server — minimal, SSH-ready
  • Desktop — full Ubuntu desktop

Boards

31 boards across multiple makers including Orange Pi, Radxa, Turing, Mixtile, and more.

Getting Started

  1. Switch to the Images tab and find your board.
  2. Download the .img.xz for your Ubuntu release and flavor.
  3. Verify the listed SHA-256 checksum.
  4. Flash with xzcat image.img.xz | sudo dd of=/dev/sdX bs=4M status=progress or use balenaEtcher.
  5. Insert into your board, power on — cloud-init handles the first boot.

Build Cadence

Images are built from a monthly refreshed rootfs (tagged YYYYMM) combined with the latest pinned kernel and U-Boot packages.
